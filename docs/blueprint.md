# **App Name**: Indilayappan Kshetram

## Core Features:

- Temple History and Info: Display detailed information about the temple's history, significance, and architecture.
- Daily Rituals Schedule: Provide an up-to-date schedule of daily rituals and ceremonies performed at the temple.
- Online Offerings: Enable devotees to make offerings (e.g., വഴിപാടുകൾ) online.
- Online Donations: Allow users to donate to the temple through a secure online portal.
- News and Events Feed: Show updates regarding the temple activities and any event or special occasion.
- Photo Gallery: Showcase a collection of images and videos related to the temple.

## Style Guidelines:

- Primary color: Traditional maroon (#800000) for a spiritual and classic feel.
- Background color: Cream (#FFFDD0), a very lightly desaturated near-yellow, for a soft, traditional background.
- Accent color: Gold (#FFD700) for highlights, borders, and important text to evoke richness and divinity.
- Body and headline font: 'Literata' (serif) for a literary and traditional look.
- Use traditional Kerala-style motifs and symbols as icons.
- Follow a layout that replicates the Guruvayur Devaswom website (guruvayurdevaswom.in), but use 'Indilayappan Kshetram' theming.
- Subtle, traditional animations to enhance user engagement (e.g., image gallery transitions).
